Kebijakan Privasi TESLA338

Kebijakan Privasi ini menjelaskan bagaimana situs informasi TESLA338 mengelola informasi yang mungkin dikumpulkan saat Anda mengunjungi halaman kami. Kami berkomitmen untuk transparansi dan perlindungan privasi pengguna kami.

1. Sifat Situs Kami (Situs Statis)

Situs ini di-*host* di GitHub Pages dan secara fundamental adalah situs statis. Artinya:

  • Kami **TIDAK** memiliki formulir input (seperti Login atau Daftar) di situs ini.
  • Kami **TIDAK** menggunakan *cookies* atau teknologi pelacakan pihak ketiga (seperti Google Analytics) untuk mengumpulkan data pribadi Anda.
  • Situs ini **TIDAK** menyimpan database pengguna.

2. Pengumpulan Data oleh Pihak Hosting (GitHub)

Sebagai platform hosting, GitHub (melalui GitHub Pages) secara otomatis mencatat informasi standar server, yang meliputi:

  • Alamat IP (Internet Protocol) pengguna.
  • Jenis peramban dan sistem operasi yang digunakan.
  • Tanggal dan waktu akses.

Pengumpulan data ini diatur sepenuhnya oleh Kebijakan Privasi GitHub dan digunakan untuk tujuan keamanan dan stabilitas hosting.

3. Tautan Eksternal dan Layanan Utama

Halaman ini menyediakan tautan ke domain utama kami (misalnya, tombol **"TESLA338 Situs Resmi"** yang mengarah ke tesla338.com).

  • Setelah Anda mengklik tautan eksternal, Anda tunduk pada kebijakan privasi dan persyaratan layanan dari domain tersebut.
  • Kami sangat menyarankan Anda untuk membaca Kebijakan Privasi di domain utama sebelum melakukan Login atau Daftar.

4. Perubahan Kebijakan Privasi

Kami dapat memperbarui Kebijakan Privasi ini dari waktu ke waktu. Setiap perubahan akan segera dipublikasikan di halaman ini dengan tanggal revisi yang jelas di bagian bawah.
